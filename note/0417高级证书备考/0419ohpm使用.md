### lottie

加载动画

```
let animationItem = lottie.loadAnimation({
   container: this.renderingContext,            // 渲染上下文
   renderer: 'canvas',                          // 渲染方式
   loop: true,         // 是否循环播放,默认true
   autoplay: true,     // 是否自动播放，默认true
   path: 'common/lottie/data.json',   // json路径
}) 
```
